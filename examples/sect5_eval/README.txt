This folder includes the experimental evaluation of AMulet 2.0.

benchmarks: 
   contains the used benchmark suites 'aoki' and 'kojevnikov'
   
experimental_data_orig:
   contains the experimental data of the evaluation that is contained in the 
   paper 
   
experimental_data_rep:
   contains scripts to reproduce our experiments.
   Please, see the general README of this artifact for an instruction how 
   to reproduce experiments.
